# Astro Winter Armor Landing Page

A modern and visually appealing landing page for the **Astro Winter Armor II** jacket. This project is built using pure **HTML5** and **CSS3**, focusing on a clean product showcase, responsive layout, and premium winter apparel branding.

## 🚀 Overview

Astro Winter Armor is a fashion-focused landing page designed to promote a premium winter jacket collection. The interface highlights product details, key features, available sizes, color variants, and call-to-action buttons for a seamless shopping experience.

## ✨ Features

* Modern and minimal UI design
* Product-centered hero section
* Interactive size selection layout
* Multiple color variant previews
* Product feature highlights
* Premium winter apparel branding
* Wishlist and Add-to-Bag buttons
* Responsive and structured layout
* Remix Icon integration for modern icons

## 🛠️ Built With

* HTML5
* CSS3
* Remix Icons
